0x01: How It Works: The Multi-Stage Infection

Silent Net doesn't just run a script; it builds an entire malicious environment on your machine in real-time to evade detection and maintain persistence. Here is the technical flow of the infection chain:

  1. The Hook: The Allatori-wrapped .jar file executes as a Minecraft mod. It immediately hooks into the Fabric/Forge entry points to grab your Session ID.
  2. Environment Check: It performs a silent check for sandboxes (Any.Run, VMs, Debuggers). If it detects a researcher environment, it terminates to avoid analysis.
  3. The Bootstrap: This is the clever part. It connects to bootstrap.pypa.io to pull down a standalone, "headless" Python runtime. This allows the malware to run complex scripts that Java-based security often misses.
  4. C2 Handshake: It queries the Polygon Blockchain via RpcHelper.java. By reading a smart contract or wallet transaction, it gets the current server IP. This makes it impossible to kill with a simple DNS block.
  5. Exfiltration: Once the Python environment is live, it starts native injections into Discord and browser processes.
