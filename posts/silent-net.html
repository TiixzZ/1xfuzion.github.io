0x02: how the bootstrap works

SNET is definitely "better" than weedhack because it doesn't just run inside Minecraft. It uses a bootstrap method to move the infection to your actual Windows system.

It hits bootstrap.pypa.io to grab a headless python environment. It basically builds a ghost version of Python in your %LOCALAPPDATA% folder. Once that’s live, it starts running scripts that your antivirus probably won't even flag because it thinks "python.exe" is just a normal program.

This is why just deleting the mod doesn't do shit. The python process is already running in the background, and it's independent of Minecraft. Even if you close the game, the R.A.T. stays alive.
