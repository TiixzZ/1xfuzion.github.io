SilentNet R.A.T: A Minecraft Rat and Stealer Targeting DonutSMP and other users.

Analysis of modular bootstrap delivery and native injections.

0x00: Introduction

SilentNet v1 is a step up from previous stealers like Weedhack due to its modular design. While it shares some DNA with the "Aseity" rigging mods found on DonutSMP, its primary function is the exfiltration of sensitive user data through a multi-stage delivery system. It's more effective than basic stealers simply because it doesn't rely solely on the Java environment to do its dirty work.

0x01: The Bootstrap Process (How it Works)

The primary danger of SilentNet is its Stage-Chain. It uses a small initial loader to bring in much larger, native tools:

  1. Execution: The .jar file hooks into Fabric/Forge.
  2. Python Deployment: It pulls a headless Python runtime from bootstrap.pypa.io. This allows the attacker to run scripts that are harder for traditional Java-based security to monitor.
  3. C2 Persistence: It uses the Polygon Blockchain (Matic) to retrieve server IPs. This is simply a more reliable way to prevent the RAT from being shut down by domain reports.

0x02: Feature Breakdown (Free vs Premium)

SilentNet is split into two tiers, gating the most invasive features behind a paywall.

The Stealer (Free Tier)

  • Discord Injection: Modifies discord_desktop_core to intercept tokens and MFA codes in real-time.
  • Session Theft: Standard extraction of Minecraft session IDs and browser cookies.

The R.A.T. (Premium Tier)

  • Exodus Injection: Targets crypto seeds and private keys.
  • Remote Control: Live webcam, screen sharing, and remote CMD access.
  • Payload Dropper: Can be used to install Worms or Ransomware as additional stages.

0x03: Removal

Step 1: Account Security

Reset passwords for Microsoft, Discord, and Email from a different device immediately.

Step 2: System Purge

Run CMD as Administrator to clear the persistent Python environment:

taskkill /F /IM python.exe /T
cd /d %LOCALAPPDATA%\Microsoft\Windows\NtProfileIndex
del /F /Q /A *
del /F /S /Q "%APPDATA%\.minecraft\mods\*.jar"

Step 3: The Better Option

Because SilentNet injects itself into local app files (like Discord), a simple file deletion might miss something. The better option is a full Windows reinstall to ensure no backdoors remain.
